第一章爬虫

1.1什么是爬虫

实质是一种计算机程序，通过计算机语言开发而来，他的行为看起来就一只蜘蛛在网上爬行工作一样，蜘蛛的英文单词：spider-爬虫的别名

百度蜘蛛，google蜘蛛实质就是一个或一套爬虫程序

爬虫就是进行数据爬取，并进行简单的数据处理（数据清理）操作的特殊应用程序。

爬虫的来源：一般通过互联网展开而来

1.2爬虫可以做什么

（1）收集数据 最直接最常用的使用方法

（2）尽职调查 是否在尽职工作 有水分可以通过数据来查看 通过“爬虫”来对发生的数据进行获取，并最终形成结果，以验证过程是否与数据所言

（3）刷流量和秒杀

1.3 爬虫的开发技术

（1）”技“ 通过各种语言和框架的使用来完成这个功能，这个层面更像是软件文档，目前市面上的教程大多停留在这个只教你如何让通过技术来实现爬虫

（2）”术” 方法 遇到反爬虫的问题，遇到一些特殊情况时，技只是实现功能的基础，而术就是提供解决问题的思路，所以更强调思想，流程调度。

目前实现爬虫的语言很多 java,php,c#,python……都能实现爬虫，其中python多采用？语法相对于其他语言简单，相对于其他语言够直接，偏脚本，语法简单，入门容易，书写方式接近英语，易看懂，有强大的库支持。

1.4使用两个库

（1）requests :请求

（2）scray :框架

第二章 python 基础

2.1 python的安装与运行

由于历史原因，目前市面上流通的python有两个版本 python2.x python3.x 区别 代码是还是3写的 比如 print(“);//python3

print”； //python2

2.1.1为什么要区分两个版本

因为两个版本之间相互不兼容 python官方宣布，在为俩python发展主要以3为主，python会新增的功能，而2主要修复，不会新增新的功能

2.1.2安装

Python 3.7.1 (也可以在官网上下载新的版本，差异不大)

windows 对应安装

mac os homebre安装：brew install tython3 在macos中已经内置python2 如果无法完成honmebre安装可以直接到官网对应的pkg安装包进行安装

linux

python-version 查看当前系统的版本 ，如果运行版本低或没有安装，或通过一下命令完成
sudo apt-update
sudo apt-get install python3.7 python3-dev python3-pip libxslt-dev zlib1g-dev libffi-devv libssl-dev

作为脚本开发语言，由于程序组成比较简单，因此编辑工具也比较简单 面向脚本的开发工具就比较多 ，比如：文本类的编辑工具都可以用于开发python，包括记事本cmd

2.3 pyhton 的数据结构和控制结构

2.3.1 两个内置的函数

print();//用于输出结果或信息，多用于在控制台输出内容

input();//用于用户输入内容

2.3.2整数、浮点、变量

（1）整数和浮点

python中的整数和数学里定义的整数是一样的，python定义的浮点和数学中定义的小数一样。在python中，我们可以通过print打印一个整数或浮点数，如 print(12345) //返回的就是整形的1234 整数的加减乘除运算可以在print中直接进行，同时还支持括号哦的方式来改变运算的优先级
在python中做除法，会得到浮点结果 在浮点运算中 0.3+0.4结果不是0.4，而是很长的一段浮点数，这不是Python问题，这是由于浮点数书的存储机制导致的。
注： 不应该直接使用python进行精确计算，但是进行日常精度不高的四则运算是没有问题的。

（2）变量

盒子，减少重复输入
一个合法的变量为等号左边为变量名，右边为变量名，其中等于就是赋值的意思
name=”zhiliaotang”
一是就是变量名为name
值为“zhiliangtang”
变量的类型完全由赋值的结果来确定，结果是整形，则为整形，结果为字符串则为字符串

补充：cmd下直接运行python脚本的方法
如脚本名为abc.py
cmd>> python abc.py

作业：完成一个简单的计算器小程序，实现简单的四则运算。要求用户输入值，程序收到值后在进行简单计算并打印结果

2.3.3字符串、列表、元组

（1）字符串

在python中字符串是一种比较常见的数据类型，任何被单引号或括号括起来的内容，都可以被认定为字符串。字符串可以赋值给变量
如：string_1=’我是字符串1’#字符串可以接收中文、英文或其他语言的字符内容
sring_2=”” #空字符串
string_3=’ ‘ #空格字符串
string_4=’a’#只有一个字母
string_5=’123’
string_6=””#采用双引号的字符串定义
字符串的内容可以说中文、英文、数字、空格以及其他字符组成。
字符串形式的数字和普通的数字是不一样的如：
int=123
string=”123” #以上变量的结果是不样的。

（2）列表（list）

数据类型的表现形式为：list=[]
列表在python中是一种容器类型，由方括号和方括号括起来的数据组成，里面的数据可以是整数、浮点数、字符串，也可以是另一个列表或其他数据结构
列表中的每一项叫做列表的一个元素，每个元素之间使用英文逗号隔开如：
list_1=[1,2,3,4]#列表中由4个元素，全是数字
list_1=[‘abc’,’x’,’知了堂’]
list_3=[]#空列表
list_4=[‘123’,[‘123’,’123’]]

（3）元组（tuple）

数据类型的表现形式为：tuple=()
元组也是python中一种容器类型，由小括号和小括号括起来的数据构成，外形和列表非常像，只不过列表为方括号，元组为小括号，”元组“中的”元“和”二元一次方程“中的元是一个意思。”组“就是组合的意思
tuple_1=(1,2,3,4,5)#6个全为数字的元素
tuple_2=(“123”,’雷雪莲’)
tuple_3=()#空元组
tuple_4=(‘123,’123,1.23,[1,2,3],[1,2,3])
以上元组的定义都是对的 其中可以发现元组支持类型>列表>字符串>整数

元组和列表的区别

列表生成后，还可以往里面继续添加数据，也可以从里面添加数据
元组生成后，就不能修改一下集中类型的数据：整数、浮点数、字符，两一个元组，里面是列表除外。

2.3.4 数据的读取

字符串、元组、列表三种的数据的读取完全一样

（1）指定下标

在大多数的编成语言里面，下标都是从0开始的 python也不例外，也是从第0个元素开始，，第0个指的是元素最左边的那个。
string name=’雷雪莲’
在字符串中，第0个字符就是’雷’，第一个是’雪’ 以此类推
list=[‘知’，’了’,’堂’]
在列表中，第0个原申诉是”知“ 以此类推
tuple=(1,2,3)
在元组中，第0个元素是1，第二个元素是’2’
变量名[下标]
来获取下标的内容

下标：范围:
0-字符长度-1，如字符串的长度为八个字符，则范围为0-7
实际上我们的取值范围为-7 -0-7
下标中，如果为正值，则是正向取值，如：
name=’我是知了堂’
name[0]=>’我’
name[1]=>’是’
name=[-1]=>’堂’

（2）切片操作

字符通过切片以后，结果还是字符串，列表切片以后还是列表，元组切片以后还是元组
为什么要切片，需要获取某个范围的内容时，需要用到切片操作
切片的格式为：
变量名[开始时位置的下标:结束位置的下标:步长]
其中开始位置的下标，结束位置的下标，步长，都可以部分省略
这三个参数对应的都是数字

范围：
开始位置的下标从零开始
结束位置的下标从0-字符串长度的-1位,不包括下标结束的内容
name=’知了堂’
name=[1:3]
name=[:3]#读取下标3以前的内容
name=[2:]#读取下标2以后的内容
步长：
1-字符串长度的-1位，或-1（-1为倒序）
步长：正值为从左到右 负值为从右到左
以上阵地字符串的且片用法，也同样适用于列表和元组

（3） 拼接和修改

字符串的拼接

元组与元组之间也是可以拼接的，相加标识两个列表拼接在一起
以上的拼接原理三个数据类型都是一样的，掌握一个，其他通用

修改内容：
字符串的额修改
字符串重新赋值即可修改
元组和列表的修改
变量名[下标]=新的值
list=[1,2,3]
list[1]=4
元组是不可变容器，列表为可变容器，所以append()智能作用域可变容器

2.3.5 字典与集合

（1）字典

字典就是用大括号括起来的键（key）值{value}对，每个键值对之间是用英文逗号隔开，每个key-value之间使用英文冒号隔开如：
dict={‘name’:’雷雪莲’,’work’:java EG}
key:中文、英文、或数据，但是是不能修改
value:可以是任意的字符串、数字、列表、元组、或另一个字典、value可以重复

字典的取值：
变量名[key]
变量名.get(key)

（2）集合

集合就是没有key的右大括号括起来的内容，而内容可以是不同的数据类型如：
set={‘1’,’2’,’3’,’c’}
集合里面的元素就不能重复的，集合也表现出了无序的特征。
集合最大的应用就是去重操作，例如，把一个带有重复元素的列表先转为集合，再转回列表，那么重复的内容元素就只会保留一个，吧列表转换为集合需要使用set()函数，把集合转换为列表用list()函数
list=[1,2,3,4,5,1,2]
uniue=list(set(list))
print(uniue)
